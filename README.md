A Java-based word game inspired by the Spelling Bee puzzle. Players form words using a fixed set of letters, always including the required center letter.

---

## Features
- Built entirely in Java using Swing for the interface  
- Validates input and prevents duplicate words  
- Tracks player score and detects pangrams  
- Simple, lightweight, and easy to run locally  

---

## Game Rules
- Use only the displayed letters to create words  
- Each word must include the center letter  
- Minimum word length is four letters  
- Using all letters forms a pangram and earns a bonus
